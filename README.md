# 🎮 Tic Tac Toe Game

A modern and interactive **Tic Tac Toe** game built using **HTML5, CSS3, and JavaScript (ES6)**. This project demonstrates core front-end development concepts such as **DOM manipulation, event handling, game state management, conditional logic, and responsive UI design**. The application delivers a smooth and engaging user experience while showcasing clean code organization and fundamental JavaScript programming skills.

---

## 🚀 Features

- 🎯 Classic two-player gameplay (Player X vs Player O)
- 🏆 Automatic winner detection
- 🤝 Draw game detection
- 🔄 Restart and New Game functionality
- ⚡ Instant move validation and game state updates
- 📱 Fully responsive design for desktop, tablet, and mobile devices
- ✨ Modern and intuitive user interface
- 🎨 Clean layout with smooth user interactions
- 🚀 Lightweight, fast, and built with pure JavaScript

---

## 🛠️ Tech Stack

- **HTML5** – Semantic page structure
- **CSS3** – Styling, layout, responsiveness, and animations
- **JavaScript (ES6)** – Game logic, DOM manipulation, and event handling

## 🎮 How to Play

1. Open the game in any modern web browser.
2. Players take turns placing **X** and **O** on the game board.
3. The first player to align three identical symbols horizontally, vertically, or diagonally wins the game.
4. If all cells are filled without a winning combination, the game ends in a draw.
5. Click **Restart** or **New Game** to begin another match.
